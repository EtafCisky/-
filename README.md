# 修仙插件 V3 - 清晰架构版本

基于 Clean Architecture 重构的修仙模拟游戏插件，为 AstrBot 提供完整的修仙玩法体验。

## ✨ 特性

- 🏗️ **清晰架构**：采用分层架构设计，代码结构清晰，易于维护
- 🎮 **完整玩法**：80+ 游戏命令，涵盖修炼、战斗、宗门、生产等多个系统
- 💾 **SQLAlchemy ORM**：使用现代 ORM 框架，数据操作安全可靠
- 🔧 **依赖注入**：松耦合设计，便于测试和扩展
- 📦 **配置化**：游戏数值通过配置文件管理，易于调整平衡性

## 🏗️ 技术架构

```text
xiuxian_v3/
├── core/                   # 核心层
│   ├── config.py          # 配置管理
│   ├── constants.py       # 常量定义
│   ├── container.py       # 依赖注入容器
│   └── exceptions.py      # 自定义异常
├── domain/                 # 领域层
│   ├── models/            # 领域模型
│   ├── enums.py           # 枚举类型
│   └── value_objects.py   # 值对象
├── application/            # 应用层
│   └── services/          # 业务服务
├── infrastructure/         # 基础设施层
│   ├── database/          # 数据库
│   └── repositories/      # 仓储
├── presentation/           # 表现层
│   └── handlers/          # 命令处理器
└── main.py                # 插件入口
```

## 📦 安装

1. 将 `xiuxian_v3` 文件夹放入 AstrBot 的 `addons` 目录
2. 重启 AstrBot
3. 发送 `/我要修仙` 开始游戏

## ⚙️ 配置

配置文件位于 `xiuxian_v3/config/` 目录：

- `level_config.json` - 法修境界配置
- `body_level_config.json` - 体修境界配置
- `weapons.json` - 武器装备配置
- `items.json` - 物品配置
- `pills.json` - 丹药配置
- `alchemy_recipes.json` - 炼丹配方
- `storage_rings.json` - 储物戒配置
- `adventure_config.json` - 历练配置
- `bounty_templates.json` - 悬赏模板

## 🔧 开发

### 添加新功能

1. 在 `domain/models/` 创建领域模型
2. 在 `infrastructure/repositories/` 创建仓储
3. 在 `application/services/` 创建业务服务
4. 在 `presentation/handlers/` 创建命令处理器
5. 在 `core/container.py` 注册依赖
6. 在 `main.py` 注册命令

### 修改游戏数值

直接编辑 `config/` 目录下的 JSON 配置文件即可。
